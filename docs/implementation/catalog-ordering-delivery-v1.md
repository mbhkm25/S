# Catalog Ordering & Delivery v1

## الهدف

تقديم تجربة طلب خفيفة داخل الملف العام للنشاط، تعتمد على واتساب بوصفه قناة الإرسال النهائية، من دون إنشاء دورة حياة طلبات دائمة داخل قاعدة البيانات في هذا الإصدار.

## النطاق المنفذ

### إعدادات إدارة النشاط

- تفعيل أو تعطيل قائمة الطلب.
- تخصيص نص زر الإضافة وزر الإرسال ومقدمة رسالة واتساب.
- التحكم في إظهار السعر والإجمالي وملاحظات العناصر.
- تحديد الحقول المطلوبة من العميل.
- اختيار نمط البطاقة والمؤثر وطريقة عرض العملة.
- اختيار عنصرين مميزين وترتيبهما.
- تمكين اختيار شركة توصيل.
- تسجيل النشاط نفسه بوصفه مقدم خدمة توصيل.
- تحديد مناطق الخدمة وأنواع التوصيل وملاحظات التسعير والتوفر.
- التحكم في مشاركة إجمالي الطلب واشتراط موافقة مشاركة بيانات العميل.

### الملف العام

- بطاقات كتالوج متكيفة مع الإعدادات.
- حالات واضحة: متاح، عند الطلب، غير متاح.
- إخفاء السعر كليًا عند تعطيل عرضه.
- تعطيل الإضافة للعناصر غير المتاحة.
- استمرار تصفح العناصر عند تعطيل قائمة الطلب.
- قائمة طلب محلية منفصلة لكل نشاط عبر `localStorage`.
- الكميات، الحذف، ملاحظات العناصر، بيانات العميل، وحالة الدفع.
- إجماليات منفصلة بحسب العملة.

### واتساب والمرجع

- مرجع واحد ثابت للطلب طوال جلسة الطلب الحالية.
- يستخدم المرجع نفسه عند الإرسال إلى المتجر أو شركة التوصيل.
- لا يتم إنشاء مرجع جديد عند كل ضغطة إرسال.
- يبدأ مرجع جديد فقط عند اختيار «بدء طلب جديد ومسح القائمة» أو عند الانتقال إلى نشاط آخر.
- روابط النشاط والمنتج تُبنى من `VITE_PUBLIC_APP_URL` عبر `urlUtils`.
- رابط الإنتاج الافتراضي: `https://app.sanadflow.com`.

### التوصيل والخصوصية

- تحميل مقدمي التوصيل المنشورين المطابقين للمحافظة والمدينة أو منطقة الخدمة.
- استبعاد النشاط نفسه من قائمة مقدمي التوصيل.
- مشاركة بيانات العميل مع مقدم التوصيل فقط بعد تحقق شروط الحقول المطلوبة.
- عند تفعيل `require_privacy_consent` لا يمكن الإرسال قبل الموافقة الصريحة.
- عند تعطيل `share_order_total` لا يظهر قسم الإجمالي في رسالة شركة التوصيل، بينما تظل بقية تفاصيل الطلب وبيانات العميل محفوظة.

## الحدود المقصودة للإصدار الأول

- لا يتم إنشاء سجل طلب دائم في قاعدة البيانات.
- لا توجد حالات قبول ورفض وتتبع داخل سند بعد.
- فتح واتساب لا يثبت أن الرسالة أُرسلت أو قُرئت.
- لا يتم مسح قائمة الطلب تلقائيًا بعد فتح واتساب.
- التوصيل في هذا الإصدار هو تنسيق مباشر بين العميل والمتجر ومقدم التوصيل عبر واتساب.
